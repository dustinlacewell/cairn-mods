## CairnDevTools v0.1.0

- Localhost HTTP console that compiles and evals C# against the live game (Roslyn).
- Scene inspection: find/inspect objects, dump the rope/belay graph, list renderers.
- Async eval (Game.WaitFor) drives the game to any state in one blocking call.
- Per-instance logging, synthetic gamepad input, and survival/edelweiss toggles for repro loops.
