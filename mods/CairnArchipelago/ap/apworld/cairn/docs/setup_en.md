# Cairn Setup Guide

## Required software

- Cairn (Steam)
- [MelonLoader](https://melonwiki.xyz) installed into the Cairn game folder
- The CairnArchipelago mod (`CairnArchipelago.dll` in `Mods/`, plus its
  libraries in `UserLibs/`)

## Configuring your YAML

A basic `Cairn` YAML works with the template generated by the Archipelago
launcher. Options:

- `death_link`: when you fall and die, everyone else with death link dies too —
  and vice versa.

## Joining a multiworld

1. Launch Cairn with MelonLoader.
2. Edit `UserData/MelonPreferences.cfg`, section `[CairnArchipelago]`: set
   `Host`, `Port`, `SlotName` and (if needed) `Password`.
3. Start or continue a save. The mod connects automatically and shows its
   status in the MelonLoader console.
